# cake3_cms_in_docker
Running CMS tutorials in Docker.

# Overview
CakePHP3系ドキュメントにあるCMSチュートリアルをDocker上で試すためのリポジトリ

# Environment
macOS Catalina 10.15.4

# Requirements
Docker version 19.03.1+
docker-compose version 1.24.1+

# Usage
### Setup
```shell script
./operation.sh setup
```

### Run
```shell script
./operation.sh
```

## Access
`http://localhost:8765/`
